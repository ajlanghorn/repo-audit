# Analyse Ministry of Justice Github repos

Scan all our repos and report on whether they're following
the rules or not.

## Usage

* `bundle install`
* [Create a Github OAuth token](https://github.com/settings/tokens/new)
* `export GH_TOKEN=[your OAuth token]`
* `bin/analyse.rb`
